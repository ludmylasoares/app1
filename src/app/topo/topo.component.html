#APP1 - Aplicação web de tradução de frases!

Curso de Desenvolvimento WEB com ES6, TypeScript e Angular

Trata-se de um jogo, onde cada frase traduzida corretamente: o progresso da barra avança! Quando a barra de progresso é totalmente preenchida o jogador vence! Caso a tradução esteja incorreta: perde-se uma tentativa/gatinho! Caso erre pela terceira vez o jogador perde!

 Processos de desenvolvimento abordados:
  • Como iniciar projetos Angular via Angular-CLI;
  • Como a SPA é iniciada no browser;
  • Como criar componentes;
  • Como integrar o Bootstrap ao projeto (para criar aplicações responsivas);
  • Databinding (string interpolation, property binding e event binding);
  • LifeCicly Hooks (ngOnInit, ngOnChanges, ngOnDestroy);
  • Diretivas (ngFor e ngIf);
  • @Input e @Output e;
  • Deploy na nuvem da AWS.
